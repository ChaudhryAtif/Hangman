# Hangman_Haskell
- A Multi-feature Hangman game written in Haskell, to be played in Console
- By Atif Chaudhry

Programming Languages Project for Saint Louis University, Spring 2015

How To Play:
- Run the program 'Hangman.hs' via Command line
